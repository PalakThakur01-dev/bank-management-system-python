# Bank Management System

## Overview

This is a simple Bank Management System developed using Python. The project allows users to create bank accounts, view account details, deposit money, withdraw money, check balances, and delete accounts through a menu-driven interface.

## Features

* Create Account
* View Account Details
* Deposit Money
* Withdraw Money
* Check Balance
* Delete Account
* Exit System

## Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements

## How to Run

```bash
python bank_management_system.py
```

## Project Structure

```text
bank_management_system.py
```
